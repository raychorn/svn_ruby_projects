cPanel® & Web Host Manager (WHM) Hosting Platform

The cPanel/WHM control panel system is the industry standard web hosting platform. It enables hosts to offer their customers an unmatched array of features. Each of your resold accounts will have its own account management interface (called "cPanel"), which provides:

  • Email system management
  • Domain management (subdomains, pointers, parking, etc)
  • Database management w/ web access interface (phpMyAdmin)
  • Site configuration and tools
  • Detailed statistics and logs
  • Pre-installed scripts and CGIs
  • A feature-rich file manager
  • Much, much, MUCH more!

The host's control panel, WHM, provides maximum flexibility and control over resold accounts. Even the most inexperienced users will find managing their hosting operations with WHM a breeze--the system is very intuitive.

For lots more info, check out the official cPanel® website.
