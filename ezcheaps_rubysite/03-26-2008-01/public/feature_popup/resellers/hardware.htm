Next Generation Reseller Hardware Platform

At EzCheapSites.Com, we realize that your website's performance is the most important part of your web hosting service. All the features in the world aren't going to help if your website is down or running slowly.

In order to ensure a rock-solid web hosting experience, we have combined our ultra low site-to-server ratios with an impressive hardware platform.

At the center of our new generation of reseller servers are two, Dual-Core Intel® Xeon processors, providing a total of four processing cores per machine. This massively powerful CPU setup is perfect for high-availability server environments.

For storage, our reseller servers utilize an RAID 5 array of 10K Raptor® SATA hard drives. This means that even if one of the hard drives in the array fails completely, the server will continue to run normally. We can replace the failed drive and the system will rebuild the data on-the-fly, without so much as a minute of downtime for your website. In addition, the use of multiple disks increases the speed at which data can be read from storage, resulting in faster page loads and database queries!

All servers have a minimum of 4 GB high-speed RAM.
