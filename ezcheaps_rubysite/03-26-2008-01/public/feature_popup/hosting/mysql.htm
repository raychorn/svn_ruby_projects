Unlimited mySQL Databases

MySQL is the web's most popular open source database engine, and powers the vast majority of web applications (blogs, forums, shopping carts and more). Applications powered by mySQL generally run much faster than those which use text files to store data.

With your account, you are allowed to create an unlimited number of mySQL databases. Databases can be managed using our control panel. You can create an unlimited number of mySQL users with different permissions, as well as modify access lists for different hostnames. This would allow you to access your database directly from your home development server, for example.

We also provide a pre-installed and configured version of phpMyAdmin, which allows you to manage all of your databases through an easy to use web interface.

For more information on mySQL, visit the project's official website.
