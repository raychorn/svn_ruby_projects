Web-Based Database Manager (phpMyAdmin)

This incredibly powerful software has become the de-facto standard for web-based MySQL database administration.

phpMyAdmin can create and drop databases, create/drop/alter tables, delete/edit/add fields, execute any SQL statement, manage keys on fields, manage privileges,export data into various formats and is available in 50 languages.

For more information on phpMyAdmin, visit the project's official website.
